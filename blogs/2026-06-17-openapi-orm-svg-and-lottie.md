A build-time codegen pipeline driven by Maven plugins generates typed Java from annotations and declarative files without reflection or runtime class loading. Features include an OpenAPI client generator, SQLite ORM, JSON/XML mapping, SVG/Lottie transcoding, a component binder with validation, and declarative routing with deep links.
